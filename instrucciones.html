Ejemplo simulado — tus datos reales pueden variar
1
Instala las reglas
Vas a copiar tokenguard.md dentro de una carpeta específica de tu proyecto. Elige una de las dos opciones:
Opción A — Recomendada
Carpeta rules
Copia a .claude/rules/ de tu proyecto. Se carga automáticamente junto a tu CLAUDE.md existente.
Opción B
Añadir a CLAUDE.md
Pega el contenido al final de tu CLAUDE.md. Útil si prefieres tenerlo todo en un solo archivo.
Sustituye tu-proyecto por la ruta real de tu proyecto. Ejemplo: si tu proyecto está en C:\dev\mi-app, el comando se convierte en mkdir -p C:\dev\mi-app\.claude\rules
# Opción A — recomendada
mkdir -p tu-proyecto/.claude/rules
cp tokenguard.md tu-proyecto/.claude/rules/
# Opción B — añadir al CLAUDE.md existente
cat tokenguard.md >> tu-proyecto/CLAUDE.md
En Windows (cmd.exe): usa mkdir tu-proyecto\.claude\rules (sin el -p) y copy tokenguard.md tu-proyecto\.claude\rules\. En PowerShell y Git Bash los comandos de arriba funcionan tal cual.
Eso es todo. Claude Code carga las reglas automáticamente en la siguiente sesión. Tu CLAUDE.md no se modifica.
2
Trabaja con normalidad
Pídele a Claude una tarea grande. Verás que deja de abrir sub-agentes innecesarios y se ciñe a tu petición. Las reglas actúan en segundo plano — no necesitas cambiar tu forma de trabajar.
Las reglas guían al agente, no a ti. Tú trabajas igual, él gasta menos.
3
Genera un informe
Después de unas cuantas sesiones, comprueba la eficiencia real de tu agente. Necesitas Python 3.x instalado (descarga desde python.org si no lo tienes). Ejecuta desde la carpeta TokenGuard:
python informe.py
El script lee los logs locales de Claude Code y genera un informe HTML. Cero tokens consumidos — es pura lectura de disco, sin llamadas a la API.
¿Dónde están los logs? El script los busca automáticamente, pero si necesitas la ruta:
Windows: abre Ejecutar (Win+R), escribe %USERPROFILE%\.claude\projects y pulsa Enter.
macOS / Linux: ~/.claude/projects/
Los informes se guardan en reports/ con la fecha:
reports/
├── report-2026-04-17.html ← el más reciente
├── report-2026-04-15.html
└── report-2026-04-12.html
Abre cualquier informe en tu navegador. Ejecútalo cuando quieras para ver datos actualizados.
Opciones
Uso avanzado
# Analizar las últimas 20 sesiones
python informe.py --sessions 20

# Filtrar por nombre de proyecto
python informe.py --project mi-app

# Ruta de salida personalizada
python informe.py --output ~/Escritorio/informe.html
